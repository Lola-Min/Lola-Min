• Tim Berners-Lee

    • Le chercheur britannique, Tim Berners-Lee a inventé le world wide Web en 1989, lorsqu'il travaillait au CERN (centre européen de recherche nucléaire).
    Tim Berners-Lee a fondé le W3C. Le W3C est un organisme a but non lucratif, qui sert à promouvoir la technologie sur le Web. Il veille à ce que le Web évolue de la meilleure façon possible. Regroupe des entreprises telles que HTML /CSS /XML /PSG.
    il propose en 1989 un projet hypertexte basé sur le programme nommé "Enquire". Enquire est un programme de stockage d'informations, qui inclut des associations. Le projet du world wide Web conçu afin d'allier les connaissances de plusieurs utilisateurs, c'est un outil de travail.
    Le travail de création du worldwideweb est mis à exécution courant, 1990 il est utilisable par le CERN À la fin de l'année, il est disponible au grand public sur Internet en 1991
    Il est actuellement le président du World Wide Web Consortium (W3C)

  • Le web

    • Le Web as été conçu pour des scientifiques et des universités afin qu'il puisse échanger instantanément.
    le Web est une sphère d'Internet, c'est une interface.
    Dans les années 1940, Vannevar Bush propose une idée de documents interconnecté. Il veut une machine capable de permettre:

    1. la visualisation de documents sur écran
    2. La localisation des documents
    3. L'association de plusieurs documents

    Le Web 1.0 peut être qualifié de bibliothèque
    le Web 2.0 est plus interactif: on participe, les informations sont partagés...

    Afin de pouvoir surfer sur le web il est nécessaire de créer un logiciel : un navigateur
    Le premier navigateur s'appelle le world wide web.
    Le 30 avril 1993 la CERN abolie ses droits dauteur sur les logiciels du web.
    La même année le premier navigateur graphique se met en place, permettant de visualiser des images.
    Ainsi le web ne cesse de se développer.
